# SkillSight – A Voice-Based Learning Platform for Visually Impaired
SkillSight is a voice-based learning platform designed to help visually impaired users learn and interact independently through speech-based technology. The platform uses Speech-to-Text (STT) and Text-to-Speech (TTS) technologies to provide a fully accessible and user-friendly learning experience without relying heavily on visual interfaces.
The system focuses on improving communication skills, career development, and interactive learning through voice-driven modules.

---

# 📌 Features

## 🎤 Voice-Based Interaction
- Speech-to-Text (STT) for user input
- Text-to-Speech (TTS) for audio responses
- Hands-free navigation and interaction

## 📖 Pronunciation Module
- Practice words and sentences
- Speech recognition-based pronunciation practice
- Real-time feedback for improvement

## 💼 Work and Career Module
- Mock interview practice
- Professional communication training
- Presentation practice
- Workplace English improvement

## 🎮 Fun Learning Zone
Interactive learning activities including:
- Word Quiz
- Math Game
- Memory Chain
- Fill the Story

## 📊 Accessibility Dashboard
- Tracks games played
- Displays total score and progress
- Helps users monitor improvement

# ⚙️ System Workflow

1. User gives voice input
2. Speech is converted into text using STT
3. Backend processes the request
4. Selected module is activated
5. System generates response
6. TTS converts response into speech
7. Audio output is provided to the user
